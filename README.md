# ICalendar-Parser

ICalendar Parser Can create calendars or work with .ics files.

*****front-end functionality*****

~Languages: HTML, CSS, JavaScript, JQuery

Description: Website uses dynamic aspects of Javascript using event 
			 listeners from JQuery to manipulate the webpage.

~Files: all in ./public

*****Software Integrations*****

~Languages: Javascript, JQuery, AJAX, Node.js

Description: JQuery pairs with Node.js and AJAX to transfer information 
			 between the client-side and server-side Javascript files. Node.js
			 also interacts with the databases.

~Files: ./public/index.js, app.js

*****Back-end functionality*****

~Langauges: C, MySQL, JSON, Javascript, Node.js

Description: C programs create an object (or struct) of valid .ics files, and converts
			 information into a parsable JSON string format. On the server side,
			 Node.js is used to create calls to MySQL to receive data from created
			 tables. This data can then be queried however on the site.
			 
~Files: (C Code):./parser, Other: ./public/index.js, ./app.js
